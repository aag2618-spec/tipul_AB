🧠 7 גישות טיפוליות עמוקות

כל גישה כוללת: רקע תיאורטי, מושגי מפתח עם תרגום עברי, כללי זיהוי ("אם X → בדוק Y"), רגישות תרבותית, סולם כמותי (1-10), ו-21 נקודות אינטגרציה בין גישות.

CBT - בק מאהלר - הפרדה-יחידנות ויניקוט - החזקה ואובייקט מעבר בולבי - התקשרות קליין - יחסי אובייקט קוהוט - פסיכולוגיית עצמי יאלום - אקזיסטנציאלי

קובץ: src/lib/therapeutic-approaches.ts (~1,300 שורות)

📍 כל כפתורי ה-AI ומיקומם
הכנה לפגישה
📄 דף פגישה → למעלה
  • כפתור: "צור הכנה לפגישה"
  • מנתח: 5 פגישות אחרונות + שאלונים מ-30 יום אחרונים
  • מה מייצר:
  • השורה התחתונה (מה הכי חשוב)
  • מה נשאר פתוח מפגישות קודמות
  • ניתוח לפי הגישה הטיפולית
  • מה לעשות / מה לא לעשות ("זהירות מ...")
  • שאלות מוצעות
  • הערכת סיכון (רקע)
  • שאלות סופרוויז'ן
ניתוח הערה מהיר
📄 דף פגישה → צד שמאל (עורך טקסט)
  • כפתור: "נתח עם AI"
  • מנתח: הערת הפגישה הנוכחית
  • מה מייצר:
  • סיכום + נושאים מרכזיים
  • ניתוח לפי גישה (Enterprise)
  • אינטגרציה בין גישות
  • Red Flags + סולמות כמותיים
  • העברה + מעברי נושא + העברה נגדית
  • הערכת סיכון + שאלות סופרוויז'ן
  • ברית טיפולית (שבר/חידוש)
ניתוח פגישה (תמציתי / מפורט) חדש
📄 דף פגישה → צד ימין (כרטיסיית ניתוח AI)
  • כפתורים: "ניתוח תמציתי" / "ניתוח מפורט"
  • מנתח: סיכום הפגישה + הקשר מטופל
  • תמציתי (250-400 מילים):
  • ניתוח ממוקד לפי גישה + אינטגרציה
  • הקשר תרבותי + הערכת סיכון
  • מפורט (500-700 מילים) - Enterprise:
  • כל מה שבתמציתי + סולמות כמותיים
  • מעברי נושא + העברה/העברה נגדית
  • ברית טיפולית + שאלות סופרוויז'ן
ניתוח שאלון בודד חובר
👤 דף מטופל → פרופיל → שאלונים
  • כפתור: "נתח" (ליד כל שאלון)
  • מנתח: תשובות השאלון + ציונים
  • מה מייצר:
  • פרשנות קלינית לפי הגישה
  • Red Flags: סתירות, תשובות "רצויות חברתית"
  • דפוסים + מה שלא נאמר
  • המלצות + שאלות לפגישה הבאה
  • הערכת סיכון + שאלות סופרוויז'ן
ניתוח כל השאלונים חובר
👤 דף מטופל → פרופיל → שאלונים
  • כפתור: "נתח את כל השאלונים"
  • מנתח: כל השאלונים של מטופל ביחד
  • מה מייצר:
  • תמונה קלינית כוללת
  • סתירות ופרדוקסים בין שאלונים
  • Red Flags + אינטגרציה בין גישות
  • סדר עדיפויות קליני + המלצות
  • הערכת סיכון + שאלות סופרוויז'ן
דוח התקדמות שודרג
👤 דף מטופל → פרופיל → שאלונים
  • כפתור: "דוח התקדמות (30 יום)"
  • מנתח: שאלונים + פגישות בטווח תאריכים
  • מה מייצר:
  • מגמות בשאלונים (שיפור/החמרה)
  • תהליך טיפולי מסיכומי פגישות
  • אינטגרציה: שאלונים מול פגישות
  • פרדוקסים: שיפור באזור + החמרה באחר
  • כל המודולים האוניברסליים
ניתוח AI מעמיק (סיכומים) שודרג
👤 דף מטופל → סיכומים → כל הסיכומים
  • כפתור: "ניתוח AI מעמיק"
  • מנתח: כל סיכומי הפגישות של המטופל
  • מה מייצר:
  • מהלך טיפול + נקודות מפנה
  • דפוסים + מה שנתקע
  • פרדוקסים + דינמיקה טיפולית
  • כל המודולים האוניברסליים
מתמלול
📄 דף פגישה → צד שמאל
  • כפתור: "מתמלול" (Sparkles icon)
  • מנתח: תמלול הקלטת פגישה
  • מה מייצר:
  • סיכום אוטומטי מהתמלול
  • ניתוח דפוסים רגשיים, התנהגותיים, יחסיים
  • ניתוח לפי גישה + המלצות
🔧 מודולים אוניברסליים (חלים על כל ניתוח)

מודולים שמוזרקים אוטומטית לכל prompt. חלקם חלים על כל סוגי הניתוח (Full) וחלקם רק על ניתוח פגישות (מסומנים).

מודול מה עושה פגישות שאלונים הכנה דוח

מעברי נושא זיהוי הימנעות, ריחוק רגשי, התנגדות ✓ — — ✓
