# Migration Instructions - Client Name Split

## שלבים לאחר ההתקנה ב-Render:

1. ההתקנה תזהה אוטומטית את השינויים בסכמה ותריץ `prisma migrate deploy`
   
2. **חשוב!** לאחר ההתקנה, יש להריץ את הפקודה הבאה ב-Render Shell כדי לפצל שמות קיימים:
   ```bash
   npx tsx scripts/migrate-client-names.ts
   ```

## מה הסקריפט עושה:

- עובר על כל המטופלים הקיימים
- מפצל את השדה `name` לשני שדות: `firstName` ו-`lastName`
- אם יש מילה אחת בשם - היא הופכת לשם פרטי ושם משפחה
- אם יש יותר ממילה - המילה הראשונה היא שם פרטי, השאר שם משפחה
- שומר את השדה `name` המקורי לתאימות לאחור

## אם יש בעיה:

הקוד תומך בגישה לשדה `name` הישן, אז המערכת תמשיך לעבוד גם ללא הרצת הסקריפט.
אבל טפסים חדשים ידרשו שם פרטי ושם משפחה נפרדים.
