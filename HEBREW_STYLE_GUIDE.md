## 🎨 כללי עיצוב

### 1. **נקודתיים**
❌ **אל תשתמש** בנקודתיים בתוויות (labels)
```
❌ סה״כ חוב:
✅ סה״כ חוב
```

### 2. **אמוג'י**
❌ **אל תשתמש** באמוג'י בהודעות מערכת אוטומטיות
```
❌ אין תשלומים ממתינים! 🎉
✅ אין תשלומים ממתינים
```
✅ **מותר** להשתמש באמוג'י בכותרות ויזואליות (UI decorative)
```
✅ 💰 רישום תשלום (כותרת עיצובית)
✅ 👤 מטופל (אייקון)
```

### 3. **סימני פיסוק**
- נקודה (.) בין משפטים נפרדים
- פסיק (,) למנוע אי בהירות בלבד
- **לא** לפני "ו" חיבורית

```
❌ הפגישה בוטלה וחויבה, מעבר לדף תשלום...
✅ הפגישה בוטלה וחויבה. מעבר לדף תשלום...
```

### 4. **ספירות וסכומים**
✅ **השאר** ספירות בטאבים ובפילטרים
```
✅ תשלומים ממתינים (12)
✅ הצג הכל (20)
✅ חובות בלבד (8)
```

✅ **השאר** סכומים בכפתורים
```
✅ תשלום כל החוב (₪200)
```

## 💼 דוגמאות מהמערכת

### ✅ הודעות הצלחה

```typescript
// ❌ לא נכון
toast.success("התשלום נרשם בהצלחה! 🎉");

// ✅ נכון
toast.success("התשלום בוצע בהצלחה");
```

### ✅ הודעות שגיאה

```typescript
// ❌ לא נכון
return NextResponse.json({ message: "לא מורשה" }, { status: 401 });

// ✅ נכון
return NextResponse.json({ message: "אין הרשאה" }, { status: 401 });
```

### ✅ תוויות (Labels)

```typescript
// ❌ לא נכון
<Label>סה״כ חוב:</Label>

// ✅ נכון
<Label>סה״כ חוב</Label>
```

### ✅ אמצעי תשלום

```typescript
// ❌ לא נכון

// ✅ נכון
```
